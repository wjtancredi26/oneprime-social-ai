# OnePrime Social AI

Projeto SaaS inicial com arquitetura profissional para desenvolvimento de uma plataforma de social AI.

## Stack

- React + Vite
- Node.js + Express
- PostgreSQL + Prisma
- Docker + Docker Compose
- Tailwind CSS

## Estrutura

- frontend/: aplicação web
- backend/: API e serviços
- database/: scripts e configuração do banco
- docker/: imagens Docker
- docs/: documentação

## Como iniciar

1. Copie o arquivo .env.example para .env
2. Instale as dependências:
   - npm install --workspaces
3. Suba o ambiente:
   - docker compose up --build

## Portas

- Frontend: http://localhost:5173
- Backend: http://localhost:3001
- PostgreSQL: http://localhost:5432
