# OnePrime Social AI

## Objetivo
Criar uma plataforma SaaS para criação de conteúdo com Inteligência Artificial e publicação automática em redes sociais.

## Tecnologias
- React
- Vite
- TailwindCSS
- Node.js
- Express
- PostgreSQL
- Prisma ORM
- Docker
- JWT
- OpenAI API
- Meta Graph API

## Funcionalidades

### Autenticação
- Login
- Cadastro
- Recuperação de senha

### Dashboard
- Visão geral
- Estatísticas

### IA
- Gerar legendas
- Gerar hashtags
- Gerar calendário mensal
- Gerar campanhas
- Gerar imagens

### Redes Sociais
- Instagram
- Facebook

### Agendamento
- Calendário
- Agendamentos automáticos

### Analytics
- Curtidas
- Comentários
- Alcance
- Crescimento
